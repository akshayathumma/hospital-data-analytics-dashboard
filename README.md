# Hospital Data Analytics Dashboard

## Project Overview
This project analyzes hospital patient data to identify disease trends, treatment costs, patient demographics, and healthcare outcomes.

## Tools Used
- Python (Data Cleaning)
- MySQL (Data Analysis)
- Tableau (Visualization)

## Key Features
- Disease distribution analysis
- Treatment cost comparison
- Age group demographics
- Patient outcome analysis
- Satisfaction analysis

## Insights
- Cancer treatments have the highest costs
- Most patients belong to middle-age group
- Recovery rate is higher than stable cases
- Certain procedures show higher satisfaction
